Equivalence of limit definitions#

A sequence $(a_n)$ in $\mathbb{R}$ converges if and only if it is Cauchy.

Proof: ($\Rightarrow$) First, let’s prove that convergent implies Cauchy:

  1. Suppose $(a_n)$ converges to $L$. Let $\varepsilon > 0$ be given.

  2. $$|a_n - L| < \frac{\varepsilon}{2} \text{ for all } n \geq N$$

  3. Then for any $m,n \geq N$:

$$ \begin{align*} |a_m - a_n| &= |a_m - L + L - a_n| \ &\leq |a_m - L| + |L - a_n| \text{ (triangle inequality)} \ &< \frac{\varepsilon}{2} + \frac{\varepsilon}{2} \ &= \varepsilon \end{align*}

$$

  1. Therefore, $(a_n)$ is Cauchy.

($\Leftarrow$) Now, let’s prove that Cauchy implies convergent:

  1. Suppose $(a_n)$ is Cauchy. We need to show it converges.

  2. $$|a_m - a_n| < 1 \text{ for all } m,n \geq N_1$$
  3. $$|a_n| \leq |a_{N_1}| + 1$$

    So the sequence is bounded.

  4. By the Bolzano-Weierstrass theorem, $(a_n)$ has a convergent subsequence $(a_{n_k})$. Let’s say it converges to $L$.

  5. Now, let $\varepsilon > 0$ be given. Choose $N$ large enough so that:

    • $|a_m - a_n| < \frac{\varepsilon}{2}$ for all $m,n \geq N$ (Cauchy property)
    • $|a_{n_k} - L| < \frac{\varepsilon}{2}$ for all $n_k \geq N$ (subsequence convergence)
  6. $$ \begin{align*} |a_n - L| &\leq |a_n - a_{n_k}| + |a_{n_k} - L| \\ &< \frac{\varepsilon}{2} + \frac{\varepsilon}{2} \\ &= \varepsilon \end{align*} $$

    where $n_k$ is chosen large enough ($\geq N$).

  7. Therefore, $(a_n)$ converges to $L$.

This completes the proof. The key insight is that:

  • Convergence ‚Üí Cauchy is relatively straightforward using the triangle inequality
  • Cauchy ‚Üí Convergence requires the completeness of $\mathbb{R}$ (via Bolzano-Weierstrass)
