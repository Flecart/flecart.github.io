With variational inference we want to find a good approximation of the posterior distribution from which it is easy to sample. The objective is to approximate the posterior with a simpler one, because sometimes the prior or likelihood are difficult to compute.

p(θ∣x1:n​,y1:n​)=z1​p(y1:n​∣θ,x1:n​)p(θ∣x1:n​)≈q(θ∣λ)

For Bayesian Linear Regression we had high dimensional Gaussians which made the inference closed form, in general this is not true, so we need some kinds of approximation.

Laplace approximation#

Introduction to the Idea#

Let's consider ψ(θ)=logp(θ∣y1:n​,x1:n​) and it's Taylor approximation (see Hopital, Taylor, Peano) around the mode θ^ for MAP. Then we have

ψ(θ)≈ψ^​(θ)=ψ(θ^)+(θ−θ^)T∇ψ(θ^)+21​(θ−θ^)THψ​(θ^)(θ−θ^)=ψ(θ^)+21​(θ−θ^)THψ​(θ^)(θ−θ^)

We simplified the term on the first order because we are considering the mode, so the gradient should be zero for the stationary point.

This Taylor approximation is quite similar to the log of a Gaussian distribution, which for a Gaussian ∼N(μ,Σ) is equal to −21​(x−μ)TΣ−1(x−μ) so we can do some sort of pattern matching and consider our psi function to be:

ψ^​(θ)=logN(θ;θ^,−Hψ−1​)+const

Then we choose q(θ)=N(θ;θ^,−Hψ−1​)∝exp(ψ^​(θ)) One can verify that the hessian in the covariance matrix is indeed symmetric semidefinite positive. (the positive definiteveness comes from the minus of a negative semidefinite matrix of the maximum point). Normally, the covariance matrix of the Laplace approximation is indicated as Λ=−Hψ−1​.

Laplace approximation for a Gaussian#

We can easily see that the Laplace approximation for a Gaussian is the same as the Gaussian itself, because the Hessian is the inverse of the covariance matrix.

It's easy to see that the mode of the Gaussian is exactly its mean, and the Hessian is computes as follows:

Dθ​Dθ​logp(θ)=(Dθ​(Σ−1μ−Σ−1θ))T=−Σ−1

An analysis of Bayesian Logistic regression#

Setting#

For bayesian logistic regression we have that

p(y1:n​∣w,x1:n​)=i=1∏N​σ(yi​wTxi​)

And that p(w)=N(w;0,σp2​I), where σ is the Sigmoid function σ(x)=1+exp(−x)1​.

We use this probability because it's natural to say

p(yi​∣w,x1:n​)={σ(wTx)1−σ(wTx)​ if y=1 if y=0​⟹σ(yi​wTx)

Thanks to the properties of the Sigmoid function 1−σ(x)=σ(−x)

MAP Weight update#

We then see that in order to have the proxy distribution for w we need to find the Hessian of that We have that (we omit x for brevity).

w^=argwmax​p(w∣y1:n​)=argwmax​logp(w)+logp(y1:n​∣w)=argwmax​2σp2​1​wTw+i=1∑N​logσ(yi​wTxi​)

Continuing:

=argwmin​−2σp2​1​wTw+i=1∑N​logp(yi​∣w)log(1+exp(−yi​wTxi​))​​⟹∂w∂logp(y1:n​∣w)​=−i=1∑N​yi​xi​σ(−yi​wTxi​)

We can interpret that last part of that Sigmoid as the probability of the error. This could then be optimized by the gradient with updates as follow:

w←w(1−2cη)+ηyxσ(−yi​wTxi​)

Laplace approximation for Bayesian logistic regression#

Then for the precision matrix we have (computation as exercise):

Λ=−∇w​∇w​logp(w∣y1:n​x1:n​)=XTdiagi∈[n]​{πi​(1−πi​)}X+σp−2​I

With πi​=σ(wTxi​). There is a nice interpretation of this covariance matrix, because if the data point is certain then the diagonal value is small, else it is a little bit larger.

Downsides of Laplace approximation#

Laplace could be overconfident, in the same way linear regression is for some samples. This is because it greedily approximates basing on the mean of it. So perhaps one direction is ok, other directions can be quite bad.

Prediction with Variational Posterior#

Now that we have an approximation for the posterior, it should be far more easier to make predictions!

p(y∗∣x∗,x1:n​,y1:n​)=∫p(y∗∣x∗,θ)p(θ∣x1:n​y1:n​)dθ≈∫p(y∗∣x∗,θ)qλ​(θ)dθ=Eθ∼qλ​​[p(y∗∣x∗,θ)]

The last could be approximated using Monte Carlo methods methods to be

Eθ∼q​[p(y∗∣x∗,θ)]≈m1​j=1∑m​p(y∗∣x∗,θ)

Or we could do a change of variables and then estimate the one dimensional integral that we have as a result with numerical methods basically to machine precision, where f=θTx

∫p(y∗∣x∗,θ)q(θ)dθ=∫σ(y∗⋅θTx)q(θ)dθ=∫σ(y∗⋅f)(x∗q(θ))df=∫σ(y∗f)N(f;x∗Tμ,x∗TΣx∗)df

As this is just a single dimensional integral, it could be well approximated with numerical methods like Gauss-Legendre quadrature.

The variational approach#

With this approach we define a family of distributions Q called variational family and the try to find the best member within this family to approximate our P.

For example we can define the variational family of Gaussians as

Q={q(θ∣λ)=N(θ;μ,Σ}

Where the parameters λ are the mean and the covariance matrix of the Gaussian.

Reverse and Forward KL#

So we optimize for this: Reverse KL:

greedily select the mode and underestimating the variance which, in this case, leads to an overconfident prediction

q∗∈argq∈Qmin​KL(q∣∣p)

Or Forward KL, this is usually preferred, but usually more difficult to compute.

q∗∈argq∈Qmin​KL(p∣∣q)

See Kullback-Leibler divergence in Entropy.

KL of Gaussians#

Understanding the properties of the KL divergence between Gaussian distributions is crucial for grasping variational inference. I strongly encourage you to review Gaussians#Information Theoretic Properties closely, probably we will use that result in the next observations

Minimizing Forward KL Divergence#

We can provide two interpretations of the forward KL divergence. One is a MLE estimation of the dataset in the variational family, the other is a moment matching.

MLE Estimation#

We can prove the following:

argλmin​KL(p∣∣q)​=argλmin​H[p∣∣qλ​]−H[p]=argλmin​Eθ∼p​[−logqλ​(y)]− const≈argλmin​−m1​j=1∑m​logqλ​(y(i))​​

The problem is that it is often unfeasible.

This tells us that any maximum likelihood estimate qλ​ minimizes the forward KL-divergence to the empirical data distribution.

The first interpretation is usually not used as we cannot draw samples using the true distribution.

Moment Matching#

We can prove that if we find such q that minimizes the forward KL divergence, then the first two moments of the distribution are the same.

First, we consider the variational family to be part of The Exponential Family. So we can write q in the following manner:

q(θ)=exp(λTT(θ)−A(λ))

Where T(θ) are the sufficient statistics of the distribution and A(λ) is the log partition function.

Then we can write the KL divergence as:

q∗=argλmax​KL(p∣∣q)​=argλmax​Eθ∼p​[logq(θ)p(θ)​]=argλmin​Eθ∼p​[λTT(θ)−A(λ)]​​

Recall that A(λ)=log∫exp(λTT(θ))dθ.

If we now differentiate and set to zero we obtain:

0​=∇λ​Eθ∼p​[λTT(θ)−A(λ)]=Eθ∼p​[T(θ)]−∇λ​A(λ)=Eθ∼p​[T(θ)]−∇λ​log∫exp(λTT(θ))dθ=Eθ∼p​[T(θ)]−Eθ∼q​[T(θ)]⟹Eθ∼p​[T(θ)]=Eθ∼q​[T(θ)]​​

Which means the expected value of the sufficient statistics of the true distribution are the same as the variational one. For Gaussians, we have that the mean and the covariance matrix are the same as those are the sufficient statistics.

Minimizing Reverse KL Divergence#

We need to compute the value

qλ∗​​=argq∈Qmin​KL(q∣∣p(⋅∣y))=argq∈Qmin​Eθ∼q​[logp(θ∣y)logq(θ)​]=argq∈Qmin​−H(q)−Eθ∼q​[logp(θ∣y)]=argq∈Qmax​H(q)+Eθ∼q​[logp(θ∣y)]=argq∈Qmax​H(q)+Eθ∼q​[logp(θ)]+Eθ∼q​[logp(y∣θ)]+const⟹qλ∗​=argq∈Qmax​Eθ∼q​[logp(y∣θ)]−KL(q∣∣p(θ))=ELBO​​

Where we have now the prior what we were trying to optimize. We can interpret the first part as sort of likelihood while the second part is a proximity measure that acts as a regularizer. This cost function is known as the variational free energy principle (highly put forward by Hinton). This optimizer has also some links with the minimum description length principle.

Evidence Lower Bound#

We have first encountered this value in Autoencoders when analyzing variational auto-encoders. We define the ELBO also in other manners, for a certain dataset Dn​, for instance, this is another formulation.

L(q,p;Dn​)=logp(y1:n​∣x1:n​)−KL(q∣∣p(⋅∣x1:n​,y1:n​))⟹logp(y1:n​)≥L(q,p;Dn​)

The evidence is defines a logp(y1:n​). This can be played with to have a lower bound on this value with the above knowledge. We can prove the lower bound in another way, noticing that:

logp(y1:n​)using Jensen​=log∫p(y1:n​∣θ)p(θ)dθ=log∫p(y1:n​∣θ)p(θ)q(θ)q(θ)​dθ≥Eθ∼q​[logp(y1:n​∣θ)q(θ)p(θ)​]=Eθ∼q​[logp(y1:n​∣θ)]−KL(q∣∣p)=ELBO​​

We observe that this is the same thing that we have seen before while trying to minimize the reverse KL divergence. Thus, finding the best approximation is equivalent to identifying the maximum lower bound on the evidence. Essentially, it involves determining the posterior that is most likely to explain the evidence.

This indicates that maximizing the evidence lower bound is an adequate method of model selection which can be used instead of maximizing the evidence (marginal likelihood) directly.

ELBO of logistic regression#

Suppose we have a classic logistic regression with prior on the weights to be p(w)=N(w;0,I) and the likelihood to be p(y∣w,x)=∏i=1N​σ(yi​wTxi​). Let's take the parameters from a Gaussian distribution N(μ,diagi∈[d]​{σi2​}), our variational family. We want now to find the ELBO for this model.

We decompose again for the likelihood and the Kullback-Leibler. We can see from Gaussians that the KL between such distributions is:

KL(q∣∣p)=21​i=1∑d​(σi2​+μi2​−logσi2​−1)

While the likelihood is:

Ew∼q​[logp(y∣w,x)]=i=1∑N​Ew∼q​[logσ(yi​wTxi​)]

And we can use everything now.

Gradient of ELBO#

One possible approach is using the score function trick, explored in RL Function Approximation. Also called score gradients or monte-carlo gradients. Another is using the so called reparametrization trick (also used for VAE in Autoencoders). We can compute the gradient of the ELBO with respect to the variational parameters λ. But the first parameter is difficult to compute.

Suppose to have a random variable ε∼ϕ and then we can write the expectation as and that θ=g(ε,λ) for invertible functions g.

Eθ∼q​[f(θ)]=Eε∼ϕ​[f(g(ε,λ))]

So now we can use a known distribution to sample from and then compute the gradient of the expectation, this allows to compute stochastic gradients. If you know the ideas of Volume, then the formula of change of variables becomes very clean.

Let's work out an example: Let's suppose q follows a Gaussian distribution N(μ,Σ) then we can reparametrize it with θ=μ+Σ1/2ε with ε∼N(0,I) that we can sample from.

∇λ​L(λ)=∇λ​Eθ∼q​[logp(y∣x,θ)]−KL(q∣∣p)=∇λ​Eε∼ϕ​[logp(y∣x,Σ1/2ε+μ)]−KL(q∣∣p)=​​

We can now take the gradient of the first term.

∇λ​Eθ∼q​[logp(y1:n​∣x1:n​,θ)]​=∇λ​Eε∼N(0,1)​[logp(y1:n​∣x1:n​,Σ1/2ε+μ)]=nEε∼N(0,1)​Ei∼unif[0,n]​[∇λ​logp(yi​∣xi​,Σ1/2ε+μ)]≈mn​j=1∑m​∇C,μ​logp(yij​​∣xij​​,Cεj​+μ)​​

Which means: We can approximate the derivate of the likelihood with respect of the variational parameters by sampling from a normal and uniformly from the dataset and only then computing an average gradient of the likelihood with respect to the parameters.

Variational inference enables us to find approximations of distributions using highly optimized stochastic optimization techniques. However, a significant drawback is the difficulty in assessing the quality of these approximations.
