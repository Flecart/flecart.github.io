<h4 id="key-properties">Key Properties<a class="anchor" aria-hidden="true" hidden href="#key-properties">#</a></h4>
<ul>
<li><strong>Linear separability</strong>: The perceptron converges if and only if the data is linearly separable (perceptron convergence theorem).</li>
<li><strong>Limitations</strong>: It cannot solve problems like XOR due to its inability to learn non-linearly separable functions.</li>
<li><strong>Extension</strong>: The <strong>multi-layer perceptron (MLP)</strong> overcomes this limitation using hidden layers and nonlinear activation functions.</li>
</ul>
<h2 id="perceptron-convergence-theorem">Perceptron Convergence Theorem<a class="anchor" aria-hidden="true" hidden href="#perceptron-convergence-theorem">#</a></h2>
<p>The perceptron learning algorithm converges in a finite number of updates if the training data is <strong>linearly separable</strong>.</p>
