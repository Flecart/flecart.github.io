4.1 Limiti finiti al finito#

4.1.1 Intorno sferico#

Dato l'insieme R si definisce l'intorno sferico aperto di x∈R di raggio r∈R l'insieme Ir​(x)=(x−r,x+r) questa nozione è molto importante per definire il limite. Lo useremo subito su un punto di accumulazione

4.1.2 Punto di accumulazione#

Un punto di accumulazione x di un insieme A⊆R è un punto tale per cui mi posso avvicinare in modo indefinito in quel punto. Infatti deve ∀r>0∈R,∃x1​∈A:x1​∈Ir​(x)∧x1​=x ossia per cui A∩Ir​(x)=∅.

Ecco che se mi avvicino in modo indefinito, possiamo definire per bene il limite tra poco.

Questo è il limite finito per una funzione

Weierstrass e Valore intermedio#

Weierstrass (Estremi finiti)#

Studia il concetto di punto di massimo o minimo assoluto. In particolare dice che esistono quei due punti per funzioni:

  1. Dominio limitato e chiuso
  2. Funzione continua

Sia data una f:[a,b]→R tale per cui sia continua. Allora esistono massimo e minimo globale per la funzione. Ossia:

∃x0​∈[a,b]:f(x)≤f(x0​),∀x∈[a,b]

E stessa cosa per il minimo.

La dimostrazione non è data. Ma è una proprietà che molti direbbero che sia intuitivamente vera. Andare a dimostrarla si entrerebbero in tecnicismi inutili secondo me. Nel caso puoi sempre approfondirla nella pagina wikipedia associata.

Quello che dice in più è che l'immagine della funzione coincide con il massimo e minimo assoluto.

Convergence Types#

Definitions#

Uniform Convergence#

The convergence is uniform if for every ϵ>0 there exists a N such that for every n>N and for every x∈A we have that ∣fn​(x)−f(x)∣<ϵ. Where fn​ is the sequence of functions and f is the limit function.

We can write the same thing with sup:

∀ϵ>0,∃N:n>N⟹x∈Asup​∣fn​(x)−f(x)∣<ϵ

Where A is the domain of the functions.

Pointwise Convergence#

The convergence is pointwise if for every x∈A and for every ϵ>0 there exists a N such that for every n>N we have that ∣fn​(x)−f(x)∣<ϵ.

Notes on the differences#

The difference is that in uniform convergence the N is the same for every x∈A, while in pointwise convergence the N can change for every x. The main implication of this difference is that uniform convergence implies pointwise convergence, but the opposite is not true.

One student could wrongfully assume that if we take the maximum of N for every x we can have uniform convergence. Then we can have a same N for every x which implies uniform convergence. But we can have a counterexample where the N is not bounded!

Pointwise convergence implies liminffn​=limfn​#

This is some random result useful for some proofs about Banach Spaces. Intuitively, if a sequence gets close to the point (pointwise convergence) then also the inferior and upper limits are the same.

Let's go step by step and rigorously show that if fn​(x)→f(x) pointwise, then

n→∞liminf​fn​(x)=n→∞lim​fn​(x).

For a sequence (fn​), we define:

n→∞liminf​fn​=m≥1sup​n≥minf​fn​. n→∞limsup​fn​=m≥1inf​n≥msup​fn​.

  • The liminf represents the largest value that the sequence eventually stays above infinitely often.

  • The limsup represents the smallest value that the sequence eventually stays below infinitely often.

  • The sequence (fn​) converges to f if and only if

    liminffn​=limsupfn​=f.

Thus, our goal is to show that if fn​(x)→f(x), then

liminffn​=f=limsupfn​.

By definition,

n→∞liminf​fn​=m≥1sup​n≥minf​fn​.

Since fn​→f, we know that for every ε>0, there exists an integer N such that for all n≥N,

fn​≥f−ε.

Taking the infimum over all n≥m for m≥N, we get

n≥minf​fn​≥f−ε.

Now, taking the supremum over all m≥1, we obtain

m≥1sup​n≥minf​fn​≥f−ε.

Since this holds for every ε>0, we conclude that

n→∞liminf​fn​≥f. n→∞limsup​fn​=m≥1inf​n≥msup​fn​.

Since fn​→f, for every ε>0, there exists an integer N such that for all n≥N,

fn​≤f+ε.

Taking the supremum over all n≥m for m≥N, we get

n≥msup​fn​≤f+ε.

Now, taking the infimum over all m≥1, we obtain

m≥1inf​n≥msup​fn​≤f+ε.

Since this holds for every ε>0, we conclude that

n→∞limsup​fn​≤f.

we now have

n→∞liminf​fn​≥f,n→∞limsup​fn​≤f.

Since liminffn​≤limsupfn​ always holds, this forces:

n→∞liminf​fn​=f=n→∞limsup​fn​.

Thus, the limit exists and is equal to f:

n→∞lim​fn​=f.

We have rigorously proven that if fn​(x)→f(x) pointwise, then

n→∞liminf​fn​(x)=n→∞lim​fn​(x).

Classical Results#

Uniform convergence implies classical convergence#

If a sequence of functions converges uniformly, then it converges pointwise.

Proof: By definition of uniform converge we have that fn​ converges uniformly to f. Then for every ϵ>0 there exists a N such that for every n>N and for every x∈A we have that ∣fn​(x)−f(x)∣<ϵ. In particular, since the inequality holds for the supremum over all x∈A, it holds for each individual x∈A. This implies that for every x∈A and for every ϵ>0 there exists a N (which is the N of uniform convergence) such that for every n>N we have that ∣fn​(x)−f(x)∣<ϵ. This is the definition of pointwise convergence. □.

Counterexample of the opposite#

Here we prove that pointwise convergence does not imply uniform convergence.

Let's consider the function fn​(x)=xn. One can prove that this function converges pointwise to

f(x)={01​if x∈[0,1)if x=1​

But there is no N such that for every point x∈[0,1] we have that ∣fn​(x)−f(x)∣<ϵ for every n>N! Let's fix that N and a ε, we show that we can find a δ>0 such that it doesn't hold for x=(1−δ). Then we have that ∣fn​(1−δ)−f(1−δ)∣=∣(1−δ)n−0∣=(1−δ)n>ε⟹δ<1−nε​, which is a contradiction.
